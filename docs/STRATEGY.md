# 我的加密货币交易经验

均线策略：

close 在 1H MA60 上方的时候做多，在 1H MA60 下方的时候做空

资金费率：

资金费率为正时倾向于做空，资金费率为负时倾向于做多

RSI 策略：

RSI 大于 70 时倾向于做空，RSI 小于 30 时倾向于做多

多空比：

多头人数占比更多时倾向于做空，空头人数占比更多时倾向于做多

OI 持仓量：

价格窄幅震荡，OI 一直增加，表明大的波动即将到来

除了趋势指标以外，考虑使用 Bollinger Bands ，ATR 等指标描述当前交易对价格的波动率，波动率过低的时候考虑按兵不动，避免被窄幅震荡反复止损

风控

我们的风控系统与当前的杠杆倍数设置无关，无论是何倍数，每次亏损超过保证金的 25% 以上时强制止损，强制止损无需经过 agent 的判断，直接执行。

动态止盈

如果仓位比较幸运开始盈利，每当收益增加保证金的 10% 以上时，推送信号给 agent，由 agent 判断是否止盈，可以动态下达止损订单来移动止损点到比开仓价格更优的位置，避免盈利回吐。

这次比赛参与的 AI agent 总共有 788 个，竞争非常激烈，所以我们考虑的交易风格是高倍杠杆+短线趋势波段交易

比特币当前价格在 92000-82000 之间波动，整体方向偏空，加密货币整体上较为弱势，所以我们整体思路以做空为主，但目前在一个宽幅震荡区间内，所以也会有部分反弹的机会，但是不要忘了大方向是偏空的。

我决定选择 cmt_dogeusdt 为策略运行的交易对，在binance它的交易对名称是 DOGEUSDT，在weex它的交易对名称是 cmt_dogeusdt，因为它的波动率相对较高，更适合短线趋势波段交易。
